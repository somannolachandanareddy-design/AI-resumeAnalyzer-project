AI Resume Analyzer-
A web application that analyzes resumes and provides insights to help candidates improve their resumes for job applications.
Built an AI-powered resume analyzer that evaluates resumes and provides ATS-based scoring and improvement suggestions.
Implemented NLP-based text analysis to extract skills and identify gaps based on job requirements.

Features
Upload your resume (PDF or DOCX) and get automated analysis.
Extracts skills, education, experience, and contact info.
Highlights missing or weak sections in your resume.
Provides suggestions to improve content and format.
Dashboard to view analysis results instantly.


Tech Stack:
React.js, Node.js, Express.js, OpenAI API, PDF Parsing


How It Works
Upload a resume via the frontend.
Frontend sends the file to the backend for parsing.
Backend extracts key information and analyzes it.
Results and suggestions are sent back to the frontend.
Users see an interactive report highlighting improvements.
